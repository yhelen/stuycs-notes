# Commune with Holy Trinity [...of data processing]

`map` in JS:

```javascript
var x = [1,2,3,4];

ar newX = x.map(function(n) {
    return n * 2;
});

// newX is [2,4,6,8]
```

`filter` in JS:
```
var x = [1,2,3,4];

var newX = x.filter(function(n) {
    return (n % 2 == 0);
});

// newX is [2,4]
```
